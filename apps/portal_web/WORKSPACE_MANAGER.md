# Workspace Manager

多工作空间管理功能，提供直观的工作空间切换、创建、编辑和删除界面。

## 功能特性

### 1. 工作空间列表
- 显示所有可用工作空间
- 区分当前工作空间和其他工作空间
- 显示工作空间名称和路径
- 支持最近使用的工作空间排序

### 2. 工作空间切换
- 点击工作空间即可快速切换
- 当前工作空间有视觉标识
- 切换后自动刷新相关数据

### 3. 工作空间创建
- 点击 "Create new workspace" 按钮
- 输入工作空间名称
- 自动生成工作空间目录
- 创建后自动切换到新工作空间

### 4. 工作空间编辑
- 点击编辑图标进入编辑模式
- 修改工作空间名称
- 支持取消和保存操作

### 5. 工作空间删除
- 点击删除图标
- 二次确认防止误删
- 删除后自动切换到其他工作空间

## 使用方法

### 打开工作空间管理器
在顶部工具栏找到工作空间选择器，点击即可打开下拉菜单。

### 切换工作空间
1. 打开工作空间管理器
2. 点击要切换的工作空间
3. 界面自动刷新并加载新工作空间的数据

### 创建新工作空间
1. 打开工作空间管理器
2. 点击 "Create new workspace" 按钮
3. 输入工作空间名称
4. 点击 ✓ 确认或按 Enter 键
5. 新工作空间创建完成并自动切换

### 重命名工作空间
1. 打开工作空间管理器
2. 找到要重命名的工作空间
3. 点击编辑图标（铅笔）
4. 修改名称
5. 点击 ✓ 保存或按 Enter 键

### 删除工作空间
1. 打开工作空间管理器
2. 找到要删除的工作空间
3. 点击删除图标（垃圾桶）
4. 再次点击确认删除
5. 工作空间被删除，自动切换到其他工作空间

## 技术实现

### 组件结构
```
WorkspaceManager
├── WorkspaceSelector (按钮)
└── WorkspaceDropdown (下拉菜单)
    ├── Header
    ├── Current Workspace Section
    ├── Other Workspaces Section
    └── Create New Section
```

### API 集成
- `GET /api/workspaces` - 获取工作空间列表
- `POST /api/workspaces` - 创建新工作空间
- `PUT /api/workspaces/{id}` - 更新工作空间
- `DELETE /api/workspaces/{id}` - 删除工作空间

### 状态管理
- 使用 TanStack Query 管理工作空间数据
- 自动缓存和重新验证
- 乐观更新提升用户体验

### 样式
- 响应式设计
- 支持深色模式
- 流畅的动画过渡
- 清晰的视觉层次

## 键盘快捷键
- `Enter` - 确认创建/编辑
- `Escape` - 取消创建/编辑
- `Tab` - 在输入框和按钮间切换

## 注意事项
1. 删除工作空间是不可逆操作，请谨慎操作
2. 工作空间名称不能为空
3. 切换工作空间会重新加载所有数据
4. 至少需要保留一个工作空间
